# 店长本地部署指南

## 📦 交付清单

给店长的文件夹应该包含：

```
attendance-system/
├── backend/                    # 后端文件
│   ├── server.js
│   ├── db.js
│   ├── routes/
│   ├── config.js
│   ├── package.json
│   └── node_modules/          (首次运行时自动生成)
├── frontend/                   # 前端文件
│   ├── src/
│   ├── package.json
│   ├── vite.config.js
│   └── node_modules/          (首次运行时自动生成)
├── attendance.db               # 数据库文件（自动生成）
├── windows-start.bat           # ← 双击启动！
├── WINDOWS_SETUP.md            # Windows 安装说明
└── README.md                   # 项目说明
```

---

## 🚀 快速启动（3步）

### 第1步：安装 Node.js

1. 访问 https://nodejs.org（选择 LTS 版本）
2. 下载安装程序
3. 双击安装，**勾选"Add Node.js to PATH"**
4. 重启电脑

验证安装：
```
node --version
```

### 第2步：解压项目文件

将收到的 `attendance-system` 文件夹解压到电脑任意位置。

建议位置：
- `C:\Users\你的用户名\Desktop\attendance-system` （桌面）
- `C:\attendance-system` （C 盘根目录）

### 第3步：双击启动

1. 进入 `attendance-system` 文件夹
2. 找到 `windows-start.bat`
3. **双击运行**

自动打开浏览器，显示登陆页面。

## 💡 第一次使用

### 登陆信息

- **网址**：http://localhost:5173
- **用户名**：admin
- **密码**：`你的管理员密码`

### 功能区域

**左上菜单：**
- 仪表盘 - 查看系统概览
- 打卡 - 员工打卡页面
- 日报 - 当天报表
- 月报 - 月度报表
- 工作时长 - 统计分析
- 店铺管理 - 添加/编辑店铺
- 员工管理 - 添加/编辑员工

**右上菜单：**
- ⚙️ 设置 - 修改密码、账户信息
- 🚪 退出 - 登出系统

## 📱 员工打卡

### 方式1：直接访问
员工访问：http://localhost:5173/checkin

### 方式2：生成二维码
在线工具生成二维码，指向打卡页面：
```
http://localhost:5173/checkin
```

员工扫码或输入员工号打卡。

## 🔒 密码管理

### 修改密码

1. 登陆系统
2. 点击右上角"⚙️ 设置"
3. 在"修改密码"部分输入：
   - 旧密码
   - 新密码（至少 6 位）
   - 确认新密码
4. 点击"修改密码"

### 忘记密码

1. 在登陆页面点击"忘记密码？"
2. 输入用户名（admin）
3. 获得临时密码
4. 用临时密码登陆后修改为新密码

## ⚙️ 日常操作

### 每天启动
1. 双击 `windows-start.bat`
2. 等待浏览器打开
3. 开始工作

### 关闭系统
1. 关闭浏览器
2. 关闭两个黑色命令窗口

### 数据保存
- 所有数据自动保存到 `attendance.db`
- 关闭系统前数据已保存，无需特殊操作

## 📊 店铺设置

### 添加店铺

1. 点击"店铺管理"
2. 点击"添加店铺"
3. 输入：
   - 店铺名称（例如：菜园子）
   - 位置/地址
   - 上班时间（例如：09:00）
   - 下班时间（例如：18:00）
4. 点击"添加"

### 编辑店铺
1. 在店铺列表中找到店铺
2. 点击"编辑"
3. 修改信息
4. 点击"保存"

### 删除店铺
1. 在店铺列表中找到店铺
2. 点击"删除"
3. 确认删除

**注意**：删除店铺前，确保没有员工属于该店铺。

---

## 👥 员工管理

### 添加单个员工
1. 点击"员工管理"
2. 点击"添加员工"
3. 输入员工信息
4. 点击"添加"

### 批量导入员工
1. 点击"批量导入"
2. 选择方式：
   - **上传 CSV**：员工号、姓名、店铺
   - **粘贴 Excel**：从 Excel 复制黏贴
3. 确认导入

### 删除员工
1. 找到员工
2. 点击"删除"
3. 确认删除

## 🔧 故障排除

### 问题1：双击 .bat 文件后无反应
**原因**：Node.js 未安装或未添加到 PATH

**解决**：
1. 重新安装 Node.js
2. 安装时勾选"Add Node.js to PATH"
3. 重启电脑
4. 再试一次

### 问题2：出现"端口被占用"错误
**原因**：3000 或 5173 端口已被占用

**解决**：
1. 关闭所有命令窗口
2. 重启电脑
3. 再试一次

### 问题3：浏览器无法打开页面
**原因**：服务器未成功启动

**解决**：
1. 查看命令窗口中的错误信息
2. 确保没有同名程序运行
3. 重启电脑

### 问题4：忘记了管理员密码
**原因**：密码遗忘

**解决**：
1. 在登陆页面点击"忘记密码？"
2. 输入 admin
3. 获得临时密码
4. 用临时密码登陆后修改密码

## 💾 数据备份

### 定期备份
1. 找到 `attendance.db` 文件
2. 复制到 U 盘或云盘
3. 按日期命名（例如：attendance-2026-04-01.db）

### 恢复数据
1. 关闭系统
2. 复制备份文件覆盖 `attendance.db`
3. 重启系统

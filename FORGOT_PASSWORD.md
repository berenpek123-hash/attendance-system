# 忘记密码功能

## 功能说明

用户忘记管理员密码时，可以通过"忘记密码"功能快速重置密码。

## 使用流程

### 1. 点击"忘记密码？"
在登陆页面，点击"忘记密码？"链接。

### 2. 输入用户名
输入你的用户名（例如 `admin`）。

### 3. 获取临时密码
系统生成一个 8 位的随机临时密码，并显示在屏幕上。

**例如：** `u23dke14`

### 4. 复制临时密码
点击"复制"按钮一键复制临时密码到剪贴板。

### 5. 用临时密码登陆
返回登陆页面，用用户名和临时密码登陆系统。

### 6. 修改密码
登陆后，进入"设置"页面，在"修改密码"部分：
- **旧密码**：输入临时密码
- **新密码**：输入新密码（至少 6 位）
- **确认新密码**：再次输入新密码

点击"修改密码"按钮。

### 7. 用新密码重新登陆
密码修改成功后，系统自动跳转到登陆页面。
用新密码登陆，进入系统。

## 技术细节

### 后端 API
```
POST /api/auth/forgot-password
请求体：{
  "username": "admin"
}

响应：{
  "success": true,
  "message": "密码已重置，请使用临时密码登陆",
  "tempPassword": "u23dke14",
  "tip": "登陆后请立即修改密码"
}
```

### 临时密码特性
- **长度**：8 位
- **字符**：数字和小写字母
- **有效期**：立即有效
- **安全建议**：登陆后立即修改为强密码

### 密码修改 API
```
POST /api/auth/change-password
请求头：Authorization: Bearer {token}
请求体：{
  "oldPassword": "临时密码",
  "newPassword": "新密码"
}

响应：{
  "success": true,
  "message": "密码修改成功"
}
```

## 安全建议

1. **立即修改**：用临时密码登陆后，立即修改为强密码
2. **强密码**：至少 8 位，包含大小写字母、数字和特殊字符
3. **不要分享**：不要将临时密码分享给他人
4. **定期更新**：定期更改密码，建议 30-90 天更新一次

## 常见问题

### Q: 临时密码可以永久使用吗？
A: 可以，但强烈建议登陆后立即修改为强密码。

### Q: 忘记了临时密码怎么办？
A: 再次点击"忘记密码？"生成新的临时密码。

### Q: 多久会过期？
A: 临时密码没有过期时间，但为了安全建议在获得后 24 小时内使用。

### Q: 可以修改别人的密码吗？
A: 不可以。只有用户本人才能修改自己的密码（需要旧密码）。

## 故障排除

### 点击"重置密码"后没反应
- 检查网络连接
- 确认用户名是否存在
- 查看浏览器控制台是否有错误

### 登陆提示密码错误
- 确认复制时没有多余空格
- 检查大小写（临时密码是小写字母+数字）
- 尝试重新生成临时密码

### 修改密码时提示旧密码错误
- 确认输入的是临时密码或修改前的密码
- 确认没有输入多余空格
- 尝试重新登陆后再修改

## 🚀 店长使用流程

### 第 1 天：初始化
1. 接收项目文件夹
2. 安装 Node.js（按 WINDOWS_SETUP.md 操作）
3. 双击 `windows-start.bat` 测试
4. 用 admin/密码 登陆验证

### 第 2-3 天：数据初始化
1. 在"店铺管理"添加所有店铺
2. 在"员工管理"批量导入员工（支持 CSV 和 Excel）
3. 设置每个店铺的营业时间（上班/下班时间）
4. 生成打卡二维码给员工

### 第 4+ 天：日常使用
1. 每天双击 `windows-start.bat` 启动
2. 员工扫码打卡
3. 查看日报/月报
4. 统计工作时长和迟到早退

## 🎓 店长需要了解的

### 核心概念
1. **打卡时间**：员工上班和下班的时间
2. **迟到判定**：打卡时间 > 店铺设置的上班时间
3. **早退判定**：下班时间 < 店铺设置的下班时间
4. **工作时长**：下班时间 - 上班时间（纯时间差）

### 日常操作顺序
1. **早上**：员工到店后扫码打卡（上班）
2. **晚上**：员工下班前扫码打卡（下班）
3. **次日**：登陆查看昨天的报表
4. **月末**：导出月报送给财务或老板

### 常见问题处理
- **员工扫码打不了卡**：检查员工号是否存在
- **显示工作时长为 0**：可能还没下班，下班后会自动更新
- **某个店铺的迟到很多**：可能需要调整店铺的上班时间
